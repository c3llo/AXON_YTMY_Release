# 개인정보처리방침

AXON YTMY Checker는 YouTube 채널의 공개 페이지를 확인해 수익 창출 여부를 추정하는 Chrome 확장 프로그램입니다.

## 수집하는 정보

이 확장 프로그램은 사용자가 현재 보고 있는 YouTube 탭의 URL 또는 사용자가 직접 입력한 YouTube 채널 URL, 핸들, 채널 ID를 처리합니다.

## 사용 목적

입력된 정보는 YouTube 공개 페이지와 공개 RSS를 조회하고, 최근 공개 영상의 수익 창출 관련 신호를 확인하는 데만 사용됩니다.

## 저장 및 전송

이 확장 프로그램은 사용자의 개인정보를 별도 서버로 전송하지 않습니다.

확장 프로그램은 검사 동작을 수행하기 위해 Chrome 브라우저에서 YouTube 공개 페이지로 직접 요청을 보냅니다. 검사 결과는 확장 프로그램 팝업 화면에 표시되며, 별도 서버에 저장되지 않습니다.

## 제3자 제공

이 확장 프로그램은 수집 또는 처리한 정보를 제3자에게 판매, 공유, 제공하지 않습니다.

## 권한 사용

이 확장 프로그램은 현재 탭의 YouTube URL을 읽고, YouTube 공개 페이지에 접근하기 위해 Chrome 확장 권한을 사용합니다. 해당 권한은 사용자가 확장 프로그램 팝업에서 확인을 실행할 때 YouTube 채널 정보를 확인하는 목적으로만 사용됩니다.
